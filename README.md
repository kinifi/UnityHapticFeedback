# UnityHapticFeedback

iOS Haptic and Notification Feedback Wrapper for Unity

### Setup

You must use the 
```
namespace UnityEngine.iOS.Haptics;
```


### API

- ``` void Create() ```
- ``` void Feedback_Success() ```
- ``` void Feedback_Warning() ```
- ``` void Feedback_Error() ```
- ``` void Impact_Light() ```
- ``` void Impact_Medium() ```
- ``` void Impact_Heavy() ```
